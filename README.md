# 🧠 Early Cognitive Decline Detection Using Voice + NeuroAI

An AI-powered web application that detects possible early cognitive decline patterns from human speech using Deep Learning and NeuroAI techniques.

---

# 🚀 Project Overview

Early symptoms of cognitive decline and Alzheimer’s disease can often appear in speech patterns such as:

- Pauses
- Reduced fluency
- Pitch variation
- Speech irregularities

This project analyzes uploaded voice recordings and predicts possible early cognitive decline using a trained Deep Learning model.

---

# ✨ Features

- 🎤 Upload voice recordings directly from the web interface
- 🧠 Deep Learning based prediction system

# 🧠 Model Information

The application uses a Deep Learning model trained on audio speech features.

### Audio Features Used
- MFCC (Mel Frequency Cepstral Coefficients)
- Speech pattern analysis
- Temporal voice characteristics

### Model Architecture
- Deep Neural Network
- Attention Mechanism
- Sequential Audio Processing

---

# 📊 Prediction Output

The system predicts:

- ✅ No Significant Cognitive Decline
OR
- ⚠️ Possible Early Cognitive Decline

along with a confidence score.
